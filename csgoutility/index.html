How to use


1.- Invite the bot to yout guild. It is strongly reccomended that you give the permissions listed automatically.

2.- Set the prefix. The default bot prefix is the "!" symbol, but you can change it with !prefix (new prefix)

3.- Sets the update news channel. By default they will be sent in the channel called "general" but if it does not exist it will not send the messages. To configure it use !config csgo_news_channel (channel name)

4.- Do you want to change the language of the bot? By default the language is English. You can change the language with !lang [language]

5.- That's it! Feel free to play around with the commands!



Commands list

!help - Displays the available commands

!config csgo_news_channel [channelName] - Configure the CSGO news bot

!language [esp|eng] - Change the language of the bot

!stats [user/steamID] - Check the CSGO statistics for any account

!prefix [prefix] - Change the bot prefix

!logros [usuario/steamID] (page) - Check out CSGO's achievements for any account

!vac [user/steamID] - Check the VAC status of an account
